# 2579번 계단 오르기 - 풀이 노트

처음에는 단순히 for문으로 가능하다고 생각했는데 총 숫자수가 몇개인지도 감이 안잡히고 분명 다른 방법이 있다고 생각해서 관련 알고리즘을 공부 후에 문제풀이 진행했습니다.

dp에 대한 문제였고 2차원 리스트에 1과 2를 저장해서 연속된 계단인지 아닌지 확인하는게 핵심이였다.

예를들어 10번째 계단을 간다고 하면 8번째 계단의 데이터에서 1연속으로 온것이든 2연속으로 온것이든 상관이 없다. 3연속은 무조건 아니기 때문이다. 따라서 이는 max를 통해 더 큰 값만 받는다.

하지만 9번째 계단일 경우에는 이야기가 다르다. 2번째로 연속으로 온 계단일 경우에는 10번째 계단까지 밟아버리면 3연속으로 밟기 때문에 규칙위반이다. 따라서 9번째 계단은 무조건 1연속으로 밟은 계단만 계산한다.

이를 표현한 점화식:

```
data[i+1][2] = score[i+1] + data[i][1]   # 9번째 계단, 인덱스 1만 계산
data[i+1][1] = max(score[i+1] + data[i-1][1], score[i+1] + data[i-1][2])   # 8번째 계단, 둘 다 상관없으므로 큰 값만 받음
```

점수의 최대를 구하는 문제이기 때문에 max를 사용한다.

사실 인덱스 에러가 떠서 다시 제출한 횟수가 많았는데, `[0, 0]`으로 data를 만들어서 인덱스 2에 대한 접근이 안됐던 것이였다. 코드를 이미 짜서 그냥 `[0, 0, 0]`으로 만들고 인덱스 0은 안 쓰는걸로 해결했다.
